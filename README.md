## How to use ?

Requirements

- FFMPEG
- NodeJs > V14. (Latest is better)

Installation
 
```bash
git clone https://github.com/ArnabXD/TGVCBot.git
cd TGVCBot
yarn
yarn build
```

Start

```bash
yarn start
```
