## How to Run ?

### Requirements ([wiki](../../wiki/Requirements))

- FFMPEG
- NodeJs > V14. (Latest is better)
- Youtube-DL
- Yarn (Optional)


### Installation
 
```bash
git clone https://github.com/ArnabXD/TGVCBot.git
cd TGVCBot
yarn
yarn build
```

### Start

```bash
yarn start
```

## Bot Usage

- `/jiosaavn <song name>` : Play Song from JioSaavn.
- `/youtube <song name/link>` : Play Song from Youtube.
- `/queue`
- `/pause`
- `/resume`
- `/skip`
- `/stopvc`
